cubeServer
==========

A distribute open source hardware testing server. 

### Features

* 1. Supports fetch config from FTP server.
* 2. Asys communication Mode.
* 3. MDNS server discovery.
* 4. Auto parse avrdue's output log.

### Dependency 

* 1. Boost
* 2. Curl
